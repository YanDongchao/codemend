Important Notes

  • numpy (as np) and matplotlib.pyplot (as plt) have been implicitly imported - There is no need to import them. It doesn't hurt to import again.
  • Any python command can be executed. There is no safety check. Run on your local server only.
  • See mpl_handler.execute() for how the code is post-processed for rendering.
